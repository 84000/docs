Replace text within one or more targets.
Requires editor.edit permission.

```graphql
replace(
  searchText: String!
  replaceText: String!
  targetUuids: [ID!]!
  type: ReplaceType = PASSAGE
  occurrenceIndex: Int
  cursorPassageUuid: ID
  cursorStart: Int
): ReplaceResult!
```
